# Prompt Test Folder

This folder contains intermediate-level prompts for testing AI/LLM abilities in three categories.

## Folder Structure

```
prompt-test/
├── coding/           # Web frontend ability tests
├── writing/          # LaTeX figure creation and PDF compilation
└── tools/            # System information collection using native tools
```

## Categories

### 1. Coding (Web Frontend Ability)
- `web-frontend-todo-app.yaml` - Build a React todo list app with CRUD operations

### 2. Writing (LaTeX + PDF)
- `latex-flowchart.yaml` - Create a flowchart using TikZ

### 3. Tools (System Information)
- `system-info-collection.yaml` - Collect comprehensive system information

## Difficulty Level: Intermediate

All prompts are designed for intermediate-level testing, requiring:
- Basic to moderate technical knowledge
- Multiple steps to complete
- Understanding of tools and their outputs
- Problem-solving skills
